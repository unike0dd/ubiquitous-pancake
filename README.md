# Gabriel Services Website

A static website for Gabriel Services, offering IT support, customer relations, logistics, and administrative services.

## Features
- Responsive design
- Dark/light theme toggle
- Bilingual (EN/ES)
- Cookie consent
- Accessible forms

## Usage
Open index.html in a browser.

## Structure
- index.html: Main page
- scripts/: JavaScript modules
- styles/: CSS files
- assets/: Images
- legal/: Static legal pages
